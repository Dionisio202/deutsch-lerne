## 1. Preguntar la hora

Hay dos formas de preguntar "¿Qué hora es?" y se responden igual:

| Alemán | Literal |
| :--- | :--- |
| **Wie spät ist es?** | ¿Qué tan tarde es? |
| **Wie viel Uhr ist es?** | ¿Cuánta hora es? *(viel = cuánto)* |

| Minutos | Expresión | Ejemplo |
| :--- | :--- | :--- |
| 1–4 min | **kurz nach** | 7:03 → *kurz nach sieben* |
| 5–29 min | **nach** | 7:10 → *zehn nach sieben* |
| :15 | **Viertel nach** | 7:15 → *Viertel nach sieben* |
| :30 | **halb** + hora siguiente | 7:30 → *halb acht* |
| :45 | **Viertel vor** | 7:45 → *Viertel vor acht* |
| 31–59 min | **vor** | 7:50 → *zehn vor acht* |
| 56–59 min | **kurz vor** | 7:58 → *kurz vor acht* |

</div>

> ⚠️ **Regla de oro del "halb":** En alemán se piensa hacia el futuro. 6:30 no es "seis y media" — es **halb sieben** (falta media hora para las **7**).

## 3. Oficial vs. Inoficial

<div style="display:grid;grid-template-columns:1fr 1fr;gap:1px;background:#d4cfc0;border-radius:8px;overflow:hidden;margin:1rem 0;">
<div style="background:#f0f5fb;padding:1.25rem;">

**🔵 Offiziell** — radio, trenes, situaciones formales

Fórmula: `[Hora] + Uhr + [Minutos]` (formato 24h)

- 14:45 → *vierzehn Uhr fünfundvierzig*
- 09:10 → *neun Uhr zehn*
- 08:30 → *acht Uhr dreißig*

</div>
<div style="background:#f0faf8;padding:1.25rem;">

**🟢 Inoffiziell** — conversación cotidiana

Fórmula: `vor / nach / halb` (formato 12h)

- 14:45 → *Viertel vor drei*
- 09:10 → *zehn nach neun*
- 08:30 → *halb neun*

</div>
</div>

## 8. Hacer y rechazar planes

**Preguntas frecuentes:**
- *Wann hast du Zeit?* → ¿Cuándo tienes tiempo?
- *Was machen wir am Mittwoch?* → ¿Qué hacemos el miércoles?
- *Wann hast du Geburtstag?* → ¿Cuándo es tu cumpleaños?

**Aceptar:**
- *Ja, das klingt super!* → ¡Sí, suena genial!
- *Um 17 Uhr ist ok.* → A las 17:00 está bien.
- *Vielleicht um 18 Uhr?* → ¿Tal vez a las 18:00?

**Rechazar:**
- *Nein, ich habe keine Zeit.* → No, no tengo tiempo.
- *Nein, das geht leider nicht.* → No, lamentablemente no se puede.
- *Nein, ich habe keine Lust.* → No, no tengo ganas.
