## Del 0 al 12 (hay que memorizar)

| Número | Alemán |
|--------|--------|
| 0 | null |
| 1 | eins |
| 2 | zwei |
| 3 | drei |
| 4 | vier |
| 5 | fünf |
| 6 | sechs |
| 7 | sieben |
| 8 | acht |
| 9 | neun |
| 10 | zehn |
| 11 | elf |
| 12 | zwölf |

## Del 13 al 19
**Regla:** número + **zehn**

| Número | Alemán | Lógica |
|--------|--------|--------|
| 13 | dreizehn | drei + zehn |
| 14 | vierzehn | vier + zehn |
| 15 | fünfzehn | fünf + zehn |
| 16 | sechzehn | sech + zehn ⚠️ |
| 17 | siebzehn | sieb + zehn ⚠️ |
| 18 | achtzehn | acht + zehn |
| 19 | neunzehn | neun + zehn |

## Las decenas
**Regla:** número + **zig**

| Número | Alemán |
|--------|--------|
| 20 | zwanzig |
| 30 | dreißig |
| 40 | vierzig |
| 50 | fünfzig |
| 60 | sechzig |
| 70 | siebzig |
| 80 | achtzig |
| 90 | neunzig |

## Del 21 al 99
**Regla:** unidad + **und** + decena

| Número | Alemán | Lógica |
|--------|--------|--------|
| 21 | einundzwanzig | ein+und+zwanzig |
| 35 | fünfunddreißig | fünf+und+dreißig |
| 47 | siebenundvierzig | sieben+und+vierzig |

## Números grandes

| Número | Alemán |
|--------|--------|
| 100 | hundert |
| 200 | zweihundert |
| 1000 | tausend |
| 1.000.000 | eine Million |

## ⚠️ Diferencia con español
En alemán los números se dicen al **revés**: primero la unidad, luego la decena.
- 47 = **sieben**und**vierzig** = "siete y cuarenta"
