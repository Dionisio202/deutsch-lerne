## Regla principal
En alemán hay dos formas de saludar: **formal** (con Sie) e **informal** (con du).


## Saludos (Grüße)

| Alemán | Español | Uso | Explicación |
|--------|---------|-----|------------------------|
| Hallo | Hola | Informal | Saludo estándar en cualquier situación informal. |
| Grüezi | Hola / Buenos días | Formal / Regional | Saludo estándar en **Suiza**. Rara vez se usa en Alemania. |
| Moin | Hola / Buenos días | Informal / Regional | Saludo del **norte de Alemania**. Se puede usar a cualquier hora del día. |
| Servus | Hola / Adiós | Informal / Regional | Se usa tanto para saludar como para despedirse en el **sur de Alemania**  |
| Guten Morgen | Buenos días | Formal / Informal | Se usa hasta las 10:00 am. Usa *Guten* porque es masculino: <span style="color: #10b981; font-weight: bold;">der</span> Morgen. |
| Guten Tag | Buenos días / Buenas tardes | Formal | Se usa aprox. de 10:00 am a 6:00 pm. Usa *Guten* porque es masculino: <span style="color: #10b981; font-weight: bold;">der</span> Tag. |
| Guten Abend | Buenas noches | Formal | Se usa a partir de las 6:00 pm para saludar. Usa *Guten* porque es masculino: <span style="color: #10b981; font-weight: bold;">der</span> Abend. |
| Gute Nacht | Buenas noches | Formal / Informal | **Solo** para ir a dormir. Usa *Gute* porque es femenino: <span style="color: #ef4444; font-weight: bold;">die</span> Nacht. |

## Decir adios (verabschieden)

| Alemán | Español | Uso | Explicación |
|--------|---------|-----|------------------------|
| Tschüss | Adiós | Informal | Es la despedida informal estándar en toda Alemania. |
| Auf Wiedersehen | Hasta luego / Adiós | Formal | Literalmente significa "Hasta volver a vernos" (*wieder* = de nuevo, *sehen* = ver). |
| Bis bald | Hasta pronto | Informal | *Bis* significa "hasta" y *bald* significa "pronto". |
| Bis morgen | Hasta mañana | Informal | *Morgen* aquí funciona como el adverbio "mañana" (al día siguiente). |
| Schönen Tag noch | Que tengas un buen día | Formal / Informal | Muy común al salir de tiendas o supermercados. Usa *Schönen* porque es: <span style="color: #10b981; font-weight: bold;">der</span> Tag. |
| Gute Nacht | Buenas noches | Al dormir | Funciona como despedida final del día antes de ir a la cama. |

## 1. Presentación y Datos Personales

| Alemán | Español | Uso / Explicación |
| :--- | :--- | :--- |
| **Wie heißt du?** | ¿Cómo te llamas? | Informal. |
| **Wie heißen Sie?** | ¿Cómo se llama? | Formal. |
| **Ich heiße...** | Me llamo... | Respuesta estándar. |
| **Woher kommst du?** | ¿De dónde eres? | Informal. Lit: "¿De dónde vienes?". |
| **Wo wohnst du?** | ¿Dónde vives? | Informal. |
| **Wie alt bist du?** | ¿Cuántos años tienes? | Informal. Lit: "¿Cuán viejo eres?". |
| **Welche Sprachen sprechen Sie?** | ¿Qué idiomas habla? | Formal. (Informal: *Welche Sprachen sprichst du?*). |
| **Ich spreche Spanisch als Muttersprache und Englisch.** | Hablo español como lengua materna e inglés. | *Muttersprache* = Lengua materna. |
| **Herr / Frau** | Señor / Señora | Se usan **siempre** acompañados del apellido (Ej: *Frau Müller*). |

## 2. Estado de Ánimo (¿Cómo estás?)

| Alemán | Español | Uso / Explicación |
| :--- | :--- | :--- |
| **Wie geht es dir?** | ¿Cómo estás? | Informal. |
| **Wie geht es Ihnen?** | ¿Cómo está usted? | Formal. |
| **Mir geht es gut.** | Estoy bien. | Respuesta correcta. Lit: "A mí me va bien". |
| **Gut, danke!** | ¡Bien, gracias! | Respuesta corta y directa. |
| **Alles gut / Alles klar** | Todo bien / Todo claro | Muy común en el día a día. |
| **Es geht** | Ahí vamos / Tirando | Cuando no estás ni bien ni mal. |
| **Na ja** | Bueno... / Más o menos | Indica duda o que las cosas no van tan bien. |

> ⚠️ **REGLA DE ORO: NUNCA DIGAS "ICH BIN GUT"**
> Para responder cómo estás, **jamás** uses "Ich bin gut". En alemán, *Ich bin gut* significa "Soy una buena persona" o "Soy bueno en algo" (ej. jugando fútbol). Para el estado de ánimo siempre debes usar **"Mir geht es gut"** (A mí me va bien).

## 3. Cortesía y Educación

| Alemán | Español | Uso / Explicación |
| :--- | :--- | :--- |
| **Danke schön** | Muchas gracias | Formal / Informal. |
| **Bitte schön / Bitte** | De nada / Por favor | *Bitte* tiene doble significado dependiendo del contexto. |
| **Sehr gern** | Con mucho gusto | Para responder a un agradecimiento de forma amable. |
| **Entschuldigung** | Disculpe / Perdón | Para llamar la atención o pedir perdón. |
| **Ich komme gleich** | Vuelvo enseguida | Lit: "Vengo pronto". |

## 4. Frases de "Supervivencia" en Clase

| Alemán | Español | Uso / Explicación |
| :--- | :--- | :--- |
| **Ich habe eine Frage.** | Tengo una pregunta. | Útil para interrumpir al profesor. |
| **Wie bitte?** | ¿Cómo? / ¿Qué ha dicho? | Más educado que decir solo "Was?" (¿Qué?). |
| **Noch einmal, bitte.** | Otra vez, por favor. | Para pedir que repitan algo. |
| **Was bedeutet...?** | ¿Qué significa...? | Ej: *Was bedeutet "Haus"?* |
| **Wie sagt man "gracias" auf Deutsch?** | ¿Cómo se dice "gracias" en alemán? | Para pedir vocabulario nuevo. |
| **Sehr gut** | Muy bien | Confirmación o elogio. |
| **Zum Beispiel (z. B.)** | Por ejemplo | Se abrevia siempre como **z. B.** (con puntos). |
