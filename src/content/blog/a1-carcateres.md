## 1. La regla oficial de reemplazo

En alemán existen 4 letras especiales. Si estás en un teclado que no las tiene o no puedes usar atajos, existe una **regla ortográfica oficial y estricta** para reemplazarlas. 

No puedes simplemente quitarle los puntos a la vocal (escribir "a" en lugar de "ä" es un error ortográfico grave que cambia el significado de la palabra). Debes añadir una **"e"** después de la vocal.

| Letra | Nombre en alemán | Reemplazo oficial | Ejemplo correcto | Ejemplo incorrecto |
| :---: | :--- | :---: | :--- | :--- |
| **ä** | *a-Umlaut* | **ae** | M**ä**dchen $\rightarrow$ M**ae**dchen | ~~Madchen~~ |
| **ö** | *o-Umlaut* | **oe** | sch**ö**n $\rightarrow$ sch**oe**n | ~~schon~~ |
| **ü** | *u-Umlaut* | **ue** | Tsch**ü**ss $\rightarrow$ Tsch**ue**ss | ~~Tschuss~~ |
| **ß** | *Eszett* (o *scharfes S*) | **ss** | Grü**ß**e $\rightarrow$ Grü**ss**e | ~~Grübe~~ |

> ⚠️ **Nota de rigor:** La letra **ß** NO es una "B" mayúscula, ni una beta griega ($\beta$). Es una doble "s" fuerte. Nunca escribas "Grübe" para decir saludos, porque "Grube" significa fosa/hoyo.

## 2. Cómo escribirlos en tu teclado (Atajos)

### En Windows (Códigos Alt)
Debes mantener presionada la tecla `Alt` y escribir el número en el teclado numérico derecho.

| Carácter | Minúscula (Alt + ...) | Mayúscula (Alt + ...) |
| :---: | :--- | :--- |
| **ä / Ä** | Alt + 132 | Alt + 142 |
| **ö / Ö** | Alt + 148 | Alt + 153 |
| **ü / Ü** | Alt + 129 | Alt + 154 |
| **ß** | Alt + 225 | *(No tiene mayúscula común)* |

### En Mac (Combinaciones de teclas)
Presiona `Option (⌥)` más la tecla indicada, suelta, y luego presiona la vocal.

| Carácter | Atajo |
| :---: | :--- |
| **ä, ö, ü** | `Option` + `u` $\rightarrow$ luego la vocal (a, o, u) |
| **ß** | `Option` + `s` |

## 3. El método más eficiente (Recomendado)

Para quienes escriben código, programan o usan comandos constantemente, cambiar el teclado a "Alemán" desconfigura símbolos críticos como `{ }`, `[ ]` o `/`. 

La solución más limpia es configurar tu sistema operativo con la distribución de teclado **"Estados Unidos - Internacional"** (US International). Esto mantiene la estructura de tu teclado intacto, pero te permite hacer las letras alemanas de forma muy rápida:

* **Para ä, ö, ü:** Presiona las comillas dobles (`"`) y luego la vocal.
* **Para ß:** Presiona la tecla `Alt Gr` (el Alt derecho) + la letra `s`.
